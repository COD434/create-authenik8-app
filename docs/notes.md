# Project notes

- create-authenik8-app generates a starter project, not a framework.
- Production deployments require external Redis for the security features that depend on durable server-side state. Local development can use the generated in-process compatible store.
- `authenik8-core` is an independently versioned MIT-licensed package.
- Generated defaults provide a secure foundation, but applications remain responsible for deployment secrets and business-level authorization.
