# Why create-authenik8-app

Developers often spend days on:

- Manual JWT setup
- Secure refresh-token handling
- Redis session configuration
- Correct access control

Authenik8 provides these foundations so you can start with a secure API or connected application instead of rebuilding identity infrastructure.

It is an auth system generator, not only an auth library. Its Identity Engine treats authentication as an identity resolution problem. It unifies credentials and OAuth providers, prevents duplicate identities, handles account linking, and normalizes provider data.

See the [Identity Engine](identity-engine.md) for the design and the [Passport.js comparison](authenik8-vs-passport.md) for a focused tradeoff table.
